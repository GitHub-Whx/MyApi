#### 简单的接口请求响应示例 ####
### 具备功能 ###
1. 具备签名认证功能
2. 数据支持 array 转 xml 和 json 格式
3. 返回的数据可进行 xml 或 json 格式解析为 array 
### 优点 ###
1. 代码简易，可读性高
2. 较完善的出错处理机制
### 待拓展 ###
1. 请求数据根据实际情况请求
2. 返回数据根据实际请求从数据库读取数据
3. 日志功能
